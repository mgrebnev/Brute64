# Brute64 

### How to use
**Linux**

     $ chmod +x brute
     ./brute -h %host_address% -l %login_list% -p %password_list% -c %success_code%
### Arguments
      -h, --help  show this help message and exit
      -host       [Target IP | domain]
      -l          [Login list]
      -p          [Password list]
      -c          [Success code. Example: 200]
      -s          [Silence mode]
      -delay      [Delay between requests]
